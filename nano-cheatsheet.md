# Nano Cheatsheet

### Search 
`Ctrl + w` - Find
`Alt + w` - Find next

Cut/Paste Operations
-----------
`ctrl + 6` | `alt + a` - Set marker (for selection)

ctrl + k - (Cut -- Line or selection)
ctrl + u - Paste text from cut operation

### Misc 
`alt + u` - Undo last operation
`ctrl + g` - Show help menu
